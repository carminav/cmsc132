Importing code from a github repository

  1. In Github, Navigate to the repository with the code you want to import into eclipse
  2. Click the "Download ZIP button" in the bottom right of the screen
  3. In eclipse -> File -> import -> choose Existing File as New Project
  4. Click browse and choose the name of the folder you want as a new project
  5. Click Finish
